# TrashTalk 幹話產生器

使用 Node.js ， Express 打造一個有趣的職業幹話產生介面。

##幹話產生

- 點選任一職業按下按紐即產生相對應的職業幹話!

#啓動方法

- 將本專案 clone 到本地端

- 進入專案資料夾

- 安裝 npm
  npm install

- 到終端機透過啓動專案
  npm run dev

- 當終端機出現以下字樣，表示伺服器成功啓動
  trash-talk server on http://localhost:3300

##開發環境

- Node.js -v10.15.0
- Express -4.17.1
- Express-Handlebars-5.3.3

#使用套件

- nodemon
- body_parser
